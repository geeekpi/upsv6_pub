# UPS v6 Usage 

## Step 1
* Download repository by using git command, open a terminal and typing following command:
```bash
cd ~
git clone https://github.com/geeekpi/upsv6_pub.git
cd upsv6_pub/
```

## Step 2
* Enable I2C function by using `raspi-config` command:
```bash
sudo raspi-config
```
